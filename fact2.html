Fun Fact

Daisy Memiliki banyak kegunaan dan simbol kerendahan hati

Selain dijadikan sebagai buket bunga atau dekoarasi ruangan, bunga daisy memiliki banyak kegunaan lain, di antaranya menjadi hiasan unik dan cantik untuk salad.

Kemudian, bunga daisy kaya vitamin C sehingga bisa dimanfaatkan untuk obat atau produk kecantikan lainnya. Bunga daisy juga dikenal dapat memperlambat pendarahan, meredakan gangguan pencernaan, dan meredakan batuk.

Adapun simbol bunga daisy putih klasik dikaitkan dengan kerendahan hati.
